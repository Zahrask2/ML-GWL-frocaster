🌊 Groundwater Level Prediction – Digital Artefact
📌 Overview

This repository contains the digital artefact developed as part of the IJC319 Responsible Data Science Project. The artefact is a web-based interface designed to present and interact with groundwater level predictions generated using machine learning and time-series models.

The project investigates the performance of SARIMAX, LSTM, and Temporal Convolutional Networks (TCNs) in predicting groundwater levels using meteorological data (temperature, precipitation, and wind speed) within the UK context.
