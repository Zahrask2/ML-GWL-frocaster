Three Approaches to Prediction

This research benchmarks the persuasion of SARIMAX, LSTM, and TCN for forecasting UK groundwater levels using historical hydro-meteorological data, and examines which meteorological variables contribute most to model performance.

S

SARIMAX

Statistical Model

Seasonal Autoregressive Integrated Moving Average with eXogenous variables. Captures linear seasonal patterns and trends through its autoregressive structure.

Long Short-Term Memory network. Learns long-range temporal dependencies through gated memory cells, capturing non-linear relationships in the time series.

Dataset

About the Data

A long-term UK observational record providing monthly groundwater measurements alongside meteorological variables.

The data is imported from

International Groundwater Resources Assessment Centre. (2025). The Global Groundwater Monitoring Network (GGMN). https://ggmn.un-igrac.org/ https://doi.org/10.58154/6Z0Y-DA34

Zippenfenig, P. (2023). Open-Meteo.com Weather API [Computer software]. Zenodo. https://doi.org/10.5281/ZENODO.7970649

Dataset Overview

  • Time Period 1944 – 2023
  • Frequency Monthly
  • Location UK
  • Target Variable Water Level (m)
  • Train / Validate / Test Split 80% / 10% / 10%

Input Features

The models use the following meteorological variables as exogenous inputs, alongside lagged groundwater level values:

  • Temperature °C (monthly avg)
  • Precipitation mm (monthly total)
  • Wind Speed m/s (monthly avg)
  • Water Level m (target / lagged)

Important Limitations

  • The study uses data from a single country, limiting the generalisability of the results to other regions or aquifer types.
  • Meteorological variables were aggregated at a monthly level without advanced lag engineering, which may reduce their predictive contribution, particularly for SARIMAX.
  • The evaluation used a single temporal train/test split. Implementing cross-validation or rolling walk-forward evaluation in future work could improve the reliability of performance estimates.
  • SARIMAX was evaluated using multi-step forecasting, generating all test predictions in a single pass, whereas LSTM and TCN used single-step-ahead forecasting with observed values at each step. This asymmetry should be considered when comparing results directly.
How It Works

Methodology Overview

A structured approach to preprocessing, building, and evaluating the three forecasting models.

01

Data Preparation

Monthly observations were cleaned, explored, and normalised using MinMaxScaler. Separate scalers for features and target ensured no data leakage from the test set into training.

02

Model Training

Each model was optimised using Bayesian hyperparameter search — Optuna (TPE) for SARIMAX and Keras Tuner for LSTM and TCN — to systematically find the best configuration.

03

Evaluation

Models were assessed on held-out test data using five metrics: RMSE, MAE, MAPE, R², and NSE.

About

About This Project

A final-year Data Science project benchmarking SARIMAX, LSTM, and TCN for groundwater prediction using meteorological data.

Project Context

This project was completed as part of the IJC319 Responsible Data Science module at the University of Sheffield. It investigates whether deep learning models (LSTM, TCN) can outperform a traditional statistical method (SARIMAX) for predicting monthly groundwater levels in the UK using meteorological data.

The research is structured around three hypotheses addressing model performance comparisons and meteorological feature importance.
