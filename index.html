Three Approaches to Prediction

This research explores the effectiveness of various time-series modelling techniques: SARIMAX, LSTM, and TCN in forecasting Groundwater Levels of the United Kingdom. We analyse how each approach leverages historical hydro-meteorological data and identify which meteorological variables significantly enhance model performance.

Dataset

About the Data

A long-term UK observational record providing monthly groundwater measurements alongside meteorological variables.

Dataset Overview

  • Time Period 1944 – 2023
  • Frequency Monthly
  • Location UK
  • Target Variable Water Level (m)
  • Train / Validation /Test Split 80% / 10% / 20%

Input Features

The models use the following meteorological variables as exogenous inputs, alongside lagged groundwater level values:

  • Temperature °C (monthly avg)
  • Precipitation mm (monthly total)
  • Wind Speed m/s (monthly avg)
  • Water Level m (target / lagged)

Important Limitations

  • The study uses data from a single region, limiting the generalizability of its results.
  • Meteorological variables were aggregated without advanced lag engineering, possibly reducing their impact on SARIMAX model accuracy.
  • The evaluation used a single temporal split. Using cross-validation in future research could improve reliability and generalisability.
  • SARIMAX was evaluated using multi-step forecasting, generating all test predictions in a single step, which may affect error accumulation. LSTM and TCN models used single-step-ahead forecasting, receiving actual observed data at each step, potentially reducing compounding errors..
How It Works

Methodology Overview

A structured approach to Preprocessing, Building, and Evaluating .

01

Data Preparation

Monthly observations were cleaned, aggregated, and normalised using MinMaxScaler. Separate scalers ensured no data leakage from test to training sets, and finally explored.

02

Model Training

Each model was optimised using Bayesian hyperparameter search — Optuna for SARIMAX and Keras Tuner for LSTM and TCN — to find the best configuration systematically.

03

Evaluation

Models were assessed on held-out test data using five standard metrics: RMSE, MAE, MAPE, R², and NSE. Naive baselines provided additional benchmarking context.

About

About This Project

A final year Data science project Benchmarking SARIMAX, LSTM, and TCN for groundwater prediction with metrological Data.

Project Context

This project was completed as part of the IJC319 Responsible Data Science module at the University of Sheffield. It investigates whether deep learning models (LSTM, TCN) can outperform traditional statistical methods (SARIMAX) for predicting monthly groundwater levels in the UK with the use of meteorological data.

The research is structured around three hypotheses addressing model performance and meteorological feature importance.
